# 🌟 Overview

**TruthGuard AI** is a modern AI-powered fake news detection platform designed to help users verify the authenticity of online news and articles.

Users simply enter a news headline or article, and the application leverages AI-based analysis to evaluate its credibility, assign a confidence score, highlight suspicious content, and provide insights into why the news may be genuine or misleading.

# 🚀 Getting Started

## 📥 Clone Repository

```bash
git clone https://github.com/Divyadharshini-03r/Truthguard-AI
```

```bash
cd truthguard-ai
```

---

## 📦 Install Dependencies

```bash
npm install
```

or

```bash
bun install
```

---

## ▶️ Run Development Server

```bash
npm run dev
```

or

```bash
bun run dev
```

Visit:

```
http://localhost:5173
```
